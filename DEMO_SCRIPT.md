## SCREEN RECORDING TIPS

- **Zoom in** when showing small text (Cmd+Plus in Chrome = zoom to 125%)
- Use **Spotlight cursor** (System Preferences → Accessibility → Pointer → Highlight) to show where you're clicking
- Record at **1920×1080** — LinkedIn and YouTube both render crisp at this resolution
- Do a **dummy run first** — 1 pass without narration, just to check click flow
- **Pause the recording** between segments if you need a moment — edits are easy
- **Add zoom crop** in editing (CapCut, DaVinci Resolve, iMovie) for the evaluation result section — the marks table is dense and benefits from a zoom-in
